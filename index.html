Q.1: Define logic gate. Classify the logic gates. (2 Marks)
Answer to Q.1:

Logic Gate (1 Mark): A logic gate is a basic building block of a digital circuit that performs a logical operation on one or more binary inputs to produce a single binary output.

Classification (1 Mark): Logic gates are classified into three types:

  1. Basic Gates: AND, OR, NOT
  2. Universal Gates: NAND, NOR
  3. Special Purpose Gates: XOR, XNOR

Q.6: What is BCD code? (2 Marks)
Answer to Q.6:

BCD (Binary Coded Decimal) is a method of representing decimal digits (0-9) in binary. In BCD, each decimal digit is represented by its equivalent 4-bit binary number. It is also known as the 8421 code.

Q.10: State and prove De Morgan's theorem (any one). (4 Marks)
Answer to Q.10:

Statement (1 Mark): De Morgan's first theorem states that the complement of a product of variables is equal to the sum of the complements of the individual variables.
(A · B)' = A' + B'

Proof (3 Marks):

The theorem can be proven using a truth table that evaluates both sides of the equation for all possible input values.

ABA'B'A·B(A·B)' (LHS)A'+B' (RHS)
0011011
0110011
1001011
1100100

Since the columns for the Left Hand Side (LHS) and Right Hand Side (RHS) are identical, the theorem is proven.

Q.12: Explain with a diagram and truth table the operation of a 2:1 multiplexer. (4 Marks)
Answer to Q.12:

A 2:1 Multiplexer (MUX) is a digital circuit that selects one of two data input lines (I₀, I₁) and forwards it to a single output line (Y). The selection is controlled by a single select line (S).

Logic Diagram (1 Mark):

Truth Table (1 Mark):

Select (S)Output (Y)
0I₀
1I₁

Operation (2 Marks): When the select line S is 0, the NOT gate inverts it to 1, enabling the top AND gate. This allows the data from input I₀ to pass to the OR gate and then to the output Y. When S is 1, the bottom AND gate is enabled, allowing data from input I₁ to pass to the output. Thus, the select line chooses which input is connected to the output.

Q.13: Perform the BCD addition: (17)₁₀ + (57)₁₀. (4 Marks)
Answer to Q.13:

Step 1: BCD Representation (1 Mark)
17 = 0001 0111
57 = 0101 0111

Step 2: Binary Addition (1 Mark)

     0001  0111   (17)
+    0101  0111   (57)
-----------------
     0110  1110

Step 3: Check and Correct Invalid Group (1 Mark)
The right group (1110) is greater than 9, so it's invalid. Add 6 (0110) to correct it.

      1110
+     0110
    --------
    1 0100

Step 4: Add Carry and Final Result (1 Mark)
Add the carry (1) to the next group: 0110 + 1 = 0111.

The final BCD result is 0111 0100, which is (74)₁₀.

Q.14: Subtract (1010)₂ - (0101)₂ using 2's complement form. (4 Marks)
Answer to Q.14:

Step 1: Find 2's Complement of Subtrahend (2 Marks)
The subtrahend is 0101.
1's complement of 0101 is 1010.
2's complement = 1's complement + 1 = 1010 + 1 = 1011.

Step 2: Add to Minuend (2 Marks)
Add the result to the minuend (1010).

      1010   (Minuend)
+     1011   (2's complement)
    --------
    1 0101

Discard the end-around carry. The result is (0101)₂.

Q.17: State AND laws. (4 Marks)
Answer to Q.17:

The fundamental laws for the Boolean AND operation are:

  1. Annulment Law (1 Mark): Any variable ANDed with 0 is 0. (A · 0 = 0)
  2. Identity Law (1 Mark): Any variable ANDed with 1 is the variable itself. (A · 1 = A)
  3. Idempotent Law (1 Mark): Any variable ANDed with itself is the variable itself. (A · A = A)
  4. Complement Law (1 Mark): Any variable ANDed with its complement is 0. (A · A' = 0)
