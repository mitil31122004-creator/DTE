Answer to Q.1:
Logic Gate

A logic gate is a fundamental building block of a digital circuit. It's an electronic device that performs a logical operation on one or more binary inputs to produce a single binary output. The output is determined by the specific logical function the gate implements (like AND, OR, NOT).

Classification of Logic Gates

Logic gates are broadly classified into three main categories:

  1. Basic Gates:
    • AND Gate: The output is '1' only if all inputs are '1'.
    • OR Gate: The output is '1' if at least one input is '1'.
    • NOT Gate (Inverter): The output is the complement of the single input.
  2. Universal Gates:
    • NAND Gate: (NOT-AND) The output is '0' only if all inputs are '1'.
    • NOR Gate: (NOT-OR) The output is '0' if at least one input is '1'.
  3. Special Purpose Gates (Arithmetic Gates):
    • XOR (Exclusive-OR) Gate: The output is '1' if the inputs are different.
    • XNOR (Exclusive-NOR) Gate: The output is '1' if the inputs are the same.

The base or radix of a number system is the total number of unique digits used to represent numbers in that system.

  • Binary: Base 2 (Digits: 0, 1)
  • Decimal: Base 10 (Digits: 0-9)
  • Octal: Base 8 (Digits: 0-7)

Answer to Q.4:

To find the 2's complement of the binary number 1010:

Step 1: Find the 1's Complement
Invert all the bits (0 → 1, 1 → 0).
Original Number: 1010
1's Complement: 0101

Step 2: Add 1 to the 1's Complement

  0101  (1's Complement)
    +    1
    ------
      0110

The 2's complement of 1010 is 0110.

Answer to Q.5:
Universal Gate: NAND Gate
Reason for being a Universal Gate

A universal gate can be used to construct any other type of logic gate (AND, OR, NOT). The NAND gate is universal because combinations of it can implement the three basic logic functions. Since all Boolean expressions can be built from these basic functions, any gate that can replicate them is considered universal.

Answer to Q.6:
BCD (Binary Coded Decimal) Code

BCD is a system of representing decimal numbers (0-9) in binary form. Each decimal digit is encoded with its own 4-bit binary equivalent. It is also known as the 8421 code because the weights of the four bits correspond to 8, 4, 2, and 1.

Example: To represent (39)₁₀ in BCD:

  • `3` is `0011`
  • `9` is `1001`

So, (39)₁₀ in BCD is `0011 1001`.

Answer to Q.13:
BCD Addition of (47)₁₀ + (57)₁₀

Step 1: Represent numbers in BCD.
47 = 0100 0111
57 = 0101 0111

Step 2: Add using binary addition.

     0100  0111   (47)
    +    0101  0111   (57)
    -----------------
         1001  1110

Step 3: Check each 4-bit group. The right group (1110) is > 9, so it's invalid.

Step 4: Correct the invalid group by adding 6 (0110).

      1110
    +     0110
        --------
        1 0100

Step 5: Add the carry (1) to the next group.

      1001
    +        1
        --------
          1010

Step 6: This result (1010) is also > 9. Correct it by adding 6.

      1010
    +     0110
        --------
        1 0000

Step 7: Combine the results with the final carry.

Final BCD: 0001 0000 0100, which is (104)₁₀.

Answer to Q.14:
Subtract (1010)₂ - (0101)₂ using 2's Complement

Perform A + (2's complement of B).

Step 1: Find 2's complement of B (0101).
1's complement: 1010
2's complement: 1010 + 1 = 1011

Step 2: Add A to the 2's complement of B.

      1010   (A)
    +     1011   (2's complement of B)
        --------
        1 0101

Step 3: Discard the end-around carry (1). The result is positive.

Result: (0101)₂.

Answer to Q.17:
Fundamental Laws of Boolean Algebra (AND Operation)
  1. Annulment Law: A · 0 = 0
  2. Identity Law: A · 1 = A
  3. Idempotent Law: A · A = A
  4. Complement Law: A · A' = 0
